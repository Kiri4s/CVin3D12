## Installation

1) install [uv](https://docs.astral.sh/uv/getting-started/installation/)

2) run:

```sh
uv sync
```

## Usage

Example for task2b:

1) set the configuration in config.yaml

2) run code:

```sh
cd src/2025.09.30/task2b/ && uv run task2b.py
```

## Requirements
